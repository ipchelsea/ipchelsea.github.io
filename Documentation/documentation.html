How to switch the background from a set

To switch the background, open each html where you want to switch the background and find the line:

<body class="bg-triangles-2">

Replace the bg-peach class with the class from the list below:

  • bg-triangles – Background with geometric shapes.
  • bg-triangles-2 – Background with geometric shapes. (Base)
  • bg-azure – Azure color
  • bg-orange – Orange color
  • bg-peach – Peach color
  • bg-green – Green color

These are the backgrounds available from the set, but you can also make your own unique background.

Contact form

In some cases, the form may not work. Contact your hosting support for problem solving.

You can also add or change form fields. To send mail using php library PHPMailer. Detailed documentation. For the form to work, a php version of 5.5 or higher is required.

To customize a contact form, you need to open the file form-contact.php from the folder php, find and replace line 78 on your emeil example@mail.com.

$mail->addAddress('example@mail.com', 'Company Name');

Here you can specify the subject of the message, line 83.

$mail->Subject = 'Contact Form';

DOM form element

  • id="" – Unique identifier for form element
  • name="" – The name of the field is intended so that the form handler can identify it
  • placeholder="" – Displays prompting text
  • required="required" – Required if the form field is required.
  • autocomplete="" – Enables or disables autocomplete: on/off
  • oninvalid="" – Error message (your text error)
<div class="form-group col-12 col-md-6">
    <i class="font-icon icon-user"></i>
    <input type="text" class="input input__icon form-control" id="nameContact" name="nameContact" placeholder="Full name" required="required" autocomplete="on" oninvalid="setCustomValidity('Fill in the field')" onkeyup="setCustomValidity('')">
    <div class="help-block with-errors"></div>
</div>
Setting on Mapbox

1. Register with Mapbox and get "Access tokens"

2. Open the mapbox.init.js file and paste your "Access Token" into the line (YOUR-ACCESS-TOKEN):

mapboxgl.accessToken = 'YOUR-ACCESS-TOKEN';

3. To configure your coordinates on the map, use the online mapbox maker
